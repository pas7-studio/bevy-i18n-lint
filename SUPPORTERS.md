## Code Contributors

Thank you to all the developers who have contributed code, bug fixes, and features to this project.

See the [Contributors](https://github.com/pas7studio/bevy-i18n-lint/graphs/contributors) page on GitHub for the full list.

## Financial Supporters

### Past Supporters

We appreciate all past supporters who have helped make this project possible.

## Other Forms of Support

We'd also like to thank:

- **Users**: Everyone who uses bevy-i18n-lint in their projects
- **Bug Reporters**: Users who take the time to report issues
- **Documentation Contributors**: Those who help improve our documentation
- **Community Members**: Everyone who participates in discussions and helps others

## Become a Supporter

There are many ways to support bevy-i18n-lint:

1. **Use it**: The best support is using the tool in your projects
2. **Report bugs**: Help us improve by reporting issues
3. **Contribute**: Submit pull requests with improvements
4. **Share**: Tell others about bevy-i18n-lint
5. **Sponsor**: Consider financial sponsorship through GitHub Sponsors
6. **Review**: Provide feedback and suggestions

Thank you for your support! ❤️
